Variables

Variables are used to store data values.

Loop

Loops are used to execute a block of code repeatedly.

Python supports both `for` and `while` loops.

for i in range(5): print(i)
Types of Loop

Python offers two main types of loops:

  • `for` loop
  • `while` loop
  • Nested loops
  • Loop with else
  • Infinite Loop
